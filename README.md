# Social Media Project
This is a social media project where you can find two models: User and Post. The project provides a set of APIs for User and Post models with user authentication.

## Requirements
Node.js
MonogoDb
React
Express.Js

## Installation
. Clone the repository to your local machine
. Navigate to the project directory
.Install the required packages => npm install 

## Authentication
The project uses token-based authentication. A user needs to login to access protected APIs by sending an access token in the Authorization header of the API request in the following format: Bearer <access-token>.

## Starting a Project 
.Go to the Client directory and Use  npm start 
.Go to the Server directory and Use  npm start
